# Idea inicial del proyecto - GymRat

## Nombre
GymRat

## Tipo de aplicación
Aplicación de consola desarrollada en Java con base de datos relacional.

## Objetivo
Desarrollar una aplicación que permita consultar ejercicios de gimnasio, clasificarlos por grupo muscular y crear rutinas personalizadas.

## Problema detectado
Muchas personas que entrenan en el gimnasio no tienen una forma clara y sencilla de guardar sus rutinas y consultar ejercicios.

## Solución propuesta
GymRat será una aplicación sencilla que permitirá almacenar ejercicios en una base de datos, buscarlos por categoría muscular y crear rutinas personalizadas adaptadas a cada usuario.

## Funcionalidades previstas
- Ver ejercicios disponibles
- Buscar ejercicios por grupo muscular
- Crear rutinas
- Añadir ejercicios a una rutina
- Consultar rutinas guardadas
- Eliminar Rutinas

## Posibles mejoras futuras
- Guardar series y repeticiones
- Añadir enlaces a vídeos de ejercicios
- Marcar ejercicios favoritos
- Registrar historial de entrenamientos
- Exportar rutinas a XML

## Alcance inicial
Se desarrollará primero una versión mínima funcional con menú por consola, conexión a base de datos mediante JDBC y documentación asociada al proyecto intermodular.
