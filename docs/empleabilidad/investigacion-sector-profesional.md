## Exploración del sector profesional

### Empresas del sector

#### 1. Ubisoft (Videojuegos)
Ubisoft es una empresa internacional dedicada al desarrollo de videojuegos como Assassin’s Creed o Far Cry.

- Qué hace: desarrollo de videojuegos y sistemas internos para gestión de juegos.
- Perfiles: desarrolladores de motores, backend, herramientas internas.
- Tecnologías: C++, motores gráficos, herramientas propias, bases de datos.

#### 2. Ableton (Producción musical)
Ableton desarrolla el software Ableton Live, utilizado para producción musical y directo.

- Qué hace: desarrollo de software de creación musical.
- Perfiles: desarrolladores de software, ingenieros de audio, backend.
- Tecnologías: C++, python o Max for live, audio digital, sistemas en tiempo real, plugins.

#### 3. Pioneer DJ (Hardware y software DJ)
Pioneer DJ desarrolla equipos de DJ como controladoras y software como Rekordbox.

- Qué hace: desarrollo de hardware y software para DJs.
- Perfiles: desarrolladores de software, firmware, sistemas embebidos.
- Tecnologías: JavaScript/TypeScript, Python, manejo de APIs, tecnologías de IA.
